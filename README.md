# x64dbg-colors-ollydbg
- ollydbg-colors
## 如何使用？
  
- 下载 __ollydbg.ini__ 到本地.
  
### 导入配置
  
1. 首先让 __x64/x32dbg__ 运行起来.
2. 应用主题 __选项 — 主题 — Dark__.
3. 更改字体大小 __选项 — 外观 — 字体__,然后将 __表格/反汇编/十进制显示/栈等__ 全部字体更改为 __新宋体/常规/10__.
4. 最后 __选项 — 导入设置...__ 选择 __ollydbg.ini__ 导入即可.
